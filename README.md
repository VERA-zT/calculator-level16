# calculator-level16
# 抖音粉丝牌 16级升级规划器

一个用于规划抖音粉丝牌 16 级升级时间的小工具。
支持指定目标升级日、自动计算每日所需亲密度、记录每日打卡进度，并提供亲密度获取方案推荐。

## 在线使用

https://vera-zt.github.io/calculator-level16/

## 功能

* 16级升级时间规划
* 派对邀请期自动计算
* 每日所需经验计算
* 每日打卡记录
* 本地数据自动保存（localStorage）
* 分享链接生成
* 亲密度获取方案推荐
* 移动端适配

## 技术栈

* HTML5
* CSS3
* Vanilla JavaScript

无需后端，无需数据库，纯前端实现。

## 使用方式

### 设置升级计划

输入：

* 当前距离16级剩余经验
* 目标升级日期
* 期望的派对邀请触发日期

系统会自动计算：

* 第一阶段（备邀期）每日目标
* 第二阶段（派对邀请期）每日目标

提醒:
请在升级前一天接受派对邀请, 注意每天亲密度增长量, 理性消费, 不要冒漾导致升级提前.

### 每日打卡

输入当天获得的亲密度后：

* 自动累计经验
* 自动更新剩余经验
* 自动重新计算每日目标

数据会保存在浏览器本地。
